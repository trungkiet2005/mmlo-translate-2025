1) Tổng quan ngắn
MMLoSo là workshop + shared task tập trung vào mô hình đa-modal / đa-ngôn ngữ cho các ngữ cảnh low-resource và ứng dụng xã hội (social impact). Có page và cuộc thi trên Kaggle/đường dẫn shared task của workshop; cuộc thi nhắm vào bài toán dịch/ghi nhận ngôn ngữ low-resource của khu vực Ấn Độ (một số ngôn ngữ Austro-asiatic / Indo-Aryan) và bài toán có yếu tố đa-modal / script khác nhau
2) Task chính của cuộc thi
Task core (shared task language challenge): xây dựng hệ thống (Unified Translation Model / MT) để dịch giữa High-Resource Languages (HRL) (Hindi / English) và một số Low-Resource Languages (LRL) (ví dụ Bhili, Mundari, Santali). Hướng chính cuộc thi tập trung LRL → HRL (ví dụ Mundari→Hindi, Santali→English, Bhili→Hindi). 
Workshop/competition cũng nhấn mạnh các đề bài phụ liên quan tới:


robustness dưới conditions thiếu dữ liệu / thiếu modality,


xử lý script khác nhau (ví dụ Santali có Ol Chiki script; English-Santali có Roman↔OlChiki issues),


và yếu tố social-impact (ứng dụng thực tế như bảo tồn ngôn ngữ, tài liệu hoá, dịch hỗ trợ cộng đồng). 
3) Dataset — cấu trúc & chi tiết (những điểm quan trọng)
Nguồn dữ liệu / con số chung: mô tả dataset shared task: ~20,000 sentence pairs cho mỗi hướng (tức ~20k cặp cho từng language-pair direction), tập train/dev được chia (ví dụ các paper dùng 95% train / 5% dev cho thí nghiệm). Ngôn ngữ/paired directions nêu trong tài liệu: Hindi↔Bhili, Hindi↔Mundari, English↔Santali; nhưng thử nghiệm chính tập trung LRL→HRL như đã nói. Santali/Ol-Chiki có lưu ý về tokenization/sequence length. arXiv+1

Submission File
For each ROW ID in the test set, you must predict the Translation for Source Sentence and save it in the Target Sentence. The submission.csv must have two columns Row ID & Target Sentence.
NOTE : Keep the name of columns as Row ID, Source Lang, Source Sentence, Target Lang & Target Sentence (watch out for case and spaces) for the successful auto-evaluation. The values in Source Lang & Target Lang must be from {Bhilli, Hindi, Mundari, Gondi, English, Santali}. Any kind of typo (even in case or space) would fail auto-evaluation. Any NULL or EMPTY value in Target Sentence must be filled with . or any other symbol. NULL values would fail evaluation as well.
